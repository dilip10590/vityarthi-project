# vityarthi-project
This is a project based on car management system.
car management system

overview:
        This project is made to choose a car based on your price,brand,year,model.
      
Features:
        You can search for a specific car by entering the brand and model and can search for the cars of same brand and cars launched on a specific year and by the            range of price and year.
      
Technologies/tools used:
        Python program is used to do this program and in the program csv files is used to save the data of the cars and user defined functions to search based on their        need.
      
Steps to install and run the project:
        First have python launcher like vs code, jupyter, python or anyother after that save the program file in your system and open it in any of the launchers and run       the program and you'll get the output           based on your input.
      
instructions for testing:
        Read the insturctions properly and provide the input correctly based on your need so that it is utilised properly.
